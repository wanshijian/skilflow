## 布局系统

### 桌面端 (≥768px)

```
Header: sticky top, 64px
内容区 padding-top: 96px
内容区 padding-x: 32px
内容区 padding-bottom: 0

全宽流式，无 max-width 限制
```

### 手机端 (<768px)

```
Header: sticky top, 64px
内容区 padding-x: 16px
底部留白: 80px (给 MobileNav)

MobileNav: fixed bottom, 56px
```

## 页面列表（供 Figma 设计清单）

1. **首页** — Hero + 输入卡片 + 类型标签 + 风格选择 + 主按钮 + 示例 + 市场入口
2. **生成页** — 流式代码 / iframe 预览 + 下载门 + 重试区
3. **文档整理** — 格式选择 + 大文本框 + 整理按钮 + 结果预览 + 下载门
4. **工具市场** — 搜索框 + 分类标签 + 排序 + 工具卡片网格 + 定制入口
5. **精品区** — 标题 + 工具卡片网格 + 定制入口
6. **工具详情** — 标题 + 类型/下载量 + iframe 预览 + 下载门 + AI 生成标记 + 定制入口
7. **我的** — 用户卡片 + 配额卡片 + 菜单列表 + 下载记录

## 如何使用此文件

### 在 Figma 里重建
1. 按上方令牌创建 Figma Color Styles / Text Styles
2. 按组件规范逐个创建组件
3. 按布局系统搭建各页面
4. 在 Figma 里优化视觉效果
5. 更新此文件（改颜色、字体、间距、组件参数）
6. 优化完成后发给我，我全部同步到代码
